jQuery Mobile Transition Effects

jQuery Mobile includes CSS3 effects that lets you choose the way a page should open.
jQuery Mobile has a variety of effects for how to transition from one page to the next.
Browser Support

The transition effect can be applied to any link or form submission by using the data-transition attribute: 
    <a href="#pageDialog" data-transition="fade">Fade to Page Dialog</a>    
    
    <a href="#pageDialog" data-transition="flip">Flip to Page Dialog</a>    
      
    <a href="#pageDialog" data-transition="flow">Flow to Page Dialog</a>    
       
    <a href="#pageDialog" data-transition="pop">Pop to Page Dialog</a>    
          
    <a href="#pageDialog" data-transition="slide">Slide to Page Dialog</a>    
            
    <a href="#pageDialog" data-transition="slidefade">SlideFade to Page Dialog</a>    
      
    <a href="#pageDialog" data-transition="slideup">SlideUp to Page Dialog</a>    
    
    <a href="#pageDialog" data-transition="slidedown">SlideDown to Page Dialog</a>    
      
    <a href="#pageDialog" data-transition="turn">Turn to Page Dialog</a>    
    
    <a href="#pageDialog" data-transition="none">None to Page Dialog</a>
All effects above also support reverse/backward actions, e.g. if you want the page to slide from left to right, instead of right to left, use the data-direction attribute with value "reverse". 
    <a href="#pageDialog" data-transition="slide" data-direction="reverse">Slide Reverse to Page Dialog</a>
