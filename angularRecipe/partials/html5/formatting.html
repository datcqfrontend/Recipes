HTML Formatting Elements

In the previous chapter, you learned about HTML styling, using the HTML style attribute.
HTML also defines special elements, for defining text with a special meaning.
HTML uses elements like <b> and <i> for formatting output, like bold or italic text.
Formatting elements were designed to display special types of text:

  • Bold text
  • Important text
  • Italic text
  • Emphasized text
  • Marked text
  • Small text
  • Deleted text
  • Inserted text
  • Subscripts
  • Superscripts

HTML Bold and Strong Formatting

The HTML <b> element defines bold text, without any extra importance.
The HTML <strong> element defines strong text, with added semantic "strong" importance.

This text is normal.

This text is bold

This text is strong

<div>
    <p>This text is normal</p>
    <p><b>This text is bold</b></p>
    <p><strong>This text is strong</strong></p>
</div>

HTML Italic and Emphasized Formatting

The HTML <i> element defines italic text, without any extra importance.
The HTML <em> element defines emphasized text, with added semantic importance.

This text is normal.

This text is bold

This text is strong

<div>
    <p>This text is normal</p>
    <p><i>This text is italic</i></p>
    <p><em>This text is emphasized</em></p>
</div>

Browsers display <strong> as <b>, and <em> as <i>.
However, there is a difference in the meaning of these tags: <b> and <i> defines bold and italic text, but <strong> and <em> means that the text is "important".

HTML Deleted Elements

The HTML <del> element defines deleted (removed) of text.

My favorite color is blue red.

<p>My favorite color is <del>blue</del> red.</p>

HTML Inserted Elements

The HTML <ins> element defines inserted (added) text.

My favorite color is red.

<p>My favorite <ins>color</ins> is red.</p>

HTML Subscript Elements

The HTML <sub> element defines subscripted text.

This is subscripted text.

This is <sub>subscripted</sub> text.

HTML Supperscript Elements

The HTML <sup> element defines superscripted text.

This is superscripted text.

This is <sup>superscripted</sup> text.
