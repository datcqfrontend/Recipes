Menus

Most web pages have some kind of a menu. In HTML, a menu is often defined in an unordered list <ul> (and styled afterwards), like this:

  • Home
  • Menu 1
  • Menu 2
  • Menu 3
<ul>
     <li><a href="#">Home</a></li>
     <li><a href="#">Menu 1</a></li>
     <li><a href="#">Menu 2</a></li>
     <li><a href="#">Menu 3</a></li>
</ul>

If you want to create a horizontal menu of the list above, add the .list-inline class to <ul>:

  • Home
  • Menu 1
  • Menu 2
  • Menu 3
<ul class="list-inline">
     <li><a href="#">Home</a></li>
     <li><a href="#">Menu 1</a></li>
     <li><a href="#">Menu 2</a></li>
     <li><a href="#">Menu 3</a></li>
</ul>

Tabs

Tabs are created with <ul class="nav nav-tabs">. Also mark the current page with <li class="active">:

  • Home
  • Menu 1
  • Menu 2
  • Menu 3

<ul class="nav nav-tabs">
     <li class="active"><a href="#">Home</a></li>
     <li><a href="#">Menu 1</a></li>
     <li><a href="#">Menu 2</a></li>
     <li><a href="#">Menu 3</a></li>
</ul>

          </ul>
     </li>
     <li><a href="#">Menu 2</a></li>
     <li><a href="#">Menu 3</a></li>
</ul>

Pills can also be displayed vertically. Just add the .nav-stacked class.

  • Menu 2
  • Menu 3

<div class="col-md-3">

Centered Tabs and Pills

To center/justify the tabs and pills, use the .nav-justified class. Note that on screens that are smaller than 768px, the list items are stacked (content will remain centered):

Dynamic Tabs

To make the tabs toggable, add the data-toggle="tab" attribute to each link. Then add a .tab-pane class with a unique ID for every tab and wrap them inside a <div> element with class .tab-content. If you want the tabs to fade in and out when clicking on them, add the .fade class to .tab-pane:

  • Menu 1
  • Menu 2

Via JavaScript

// Select all tabs
$('.nav-tabs a').click(function(){
     $(this).tab('show');
})

// Select tab by name
$('.nav-tabs a[href="#home"]').tab('show')

// Select first tab
$('.nav-tabs a:first').tab('show')

// Select last tab
$('.nav-tabs a:last').tab('show')

// Select fourth tab (zero-based)
$('.nav-tabs li:eq(3) a').tab('show')

Tab Methods

Method Description
.tab("show") Shows the tab

Tab Events

Event Description
show.bs.tab Occurs when the tab is about to be shown.
shown.bs.tab Occurs when the tab is fully shown (after CSS transitions have completed)
hide.bs.tab Occurs when the tab is about to be hidden
hidden.bs.tab Occurs when the tab is fully hidden (after CSS transitions have completed)
