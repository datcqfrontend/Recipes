AngularJS $http

AngularJS $http is a core service for reading data from web servers.

$http.get(url) is the function to use for reading server data.

<div ng-app="myApp" ng-controller="customersCtrl">
      <ul>
           <li ng-repeat="x in names">
                { { x.Name + ', ' + x.Country } }
           </li>
      </ul>
</div>

<script>
      var app = angular.module('myApp', []);
      app.controller('customersCtrl', function($scope, $http) {
           $http.get("http://www.w3schools.com/angular/customers.php")
           .success(function(response) {
               $scope.names = response.records;
          });
      });
</script>

AngularJS $http & json

Use $http to load local json data like below:

  • {{ x.Name + ', ' + x.Country }}

<div ng-app="myApp" ng-controller="demoCustomerCtrl">
      <ul>
           <li ng-repeat="x in names">
                { { x.Name + ', ' + x.Country } }
           </li>
      </ul>
</div>

<script>
      var app = angular.module('myApp', []);
      app.controller('demoCustomerCtrl', function($scope, $http) {
           $http.get("partials/angular/customer.json")
           .success(function(response) {
               $scope.names = response.records;
          });
      });
</script>

Application explained:


The AngularJS application is defined by ng-app. The application runs inside a <div>.
The ng-controller directive names the controller object.
The demoCustomerCtrl function is a standard JavaScript object constructor.
AngularJS will invoke demoCustomerCtrl with a $scope and $http object.
$scope is the application object (the owner of application variables and functions).
$http is an XMLHttpRequest object for requesting external data.
$http.get() reads JSON data from "partials/angular/customer.json".
If success, the controller creates a property (names) in the scope, with JSON data from the server.
