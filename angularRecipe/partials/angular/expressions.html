AngularJS Expressions

AngularJS expressions are written inside double braces: {{ expression }}.
AngularJS expressions binds data to HTML the same way as the ng-bind directive.
AngularJS will "output" data exactly where the expression is written.
AngularJS expressions are much like JavaScript expressions: They can contain literals, operators, and variables.
Example { { 5 + 5 } } or { { firstName + " " + lastName } }

My first expression: {{ 5 + 5 }}


<p>My first expression: { { 5 + 5 } }</p>

AngularJS Numbers

AngularJS numbers are like JavaScript numbers:

Total in dollar (expresion): {{ quantity * cost }}

Total in dollar (ng-bind):


<p ng-init="quantity=2;cost=5">
     Total in dollar (expresion): { { quantity * cost } }
</p>
<p ng-init="quantity=2;cost=5">
     Total in dollar (ng-bind): <span ng-bind="quantity * cost"></span>
</p>

AngularJS Strings

AngularJS strings are like JavaScript strings:

The name is (expression:) {{ firstName + " " + lastName }}

The name is (ng-bind):


<p ng-init="firstName='John';lastName='Doe'">
     The name is (expresion): { { firstName + " " + lastName } }
</p>
<p ng-init="firstName='John';lastName='Doe'">
     The name is (ng-bind): <span ng-bind="firstName + ' ' + lastName"></span>
</p>

AngularJS objects are like JavaScript objects:

The name is (expression:) {{ person.firstName + " " + person.lastName }}

The name is (ng-bind):


<p ng-init="person={firstName:'John',lastName:'Doe'}">
     The name is (expresion): { { person.firstName + " " + person.lastName } }
</p>
<p ng-init="person={firstName:'John',lastName:'Doe'}">
     The name is (ng-bind): <span ng-bind="person.firstName + ' ' + person.lastName"></span>
</p>

<p ng-init="points=[1,15,19,2,40]">

</p>
<p ng-init="points=[1,15,19,2,40]">

AngularJS Expressions vs. JavaScript Expressions

Like JavaScript expressions, AngularJS expressions can contain literals, operators, and variables.
Unlike JavaScript expressions, AngularJS expressions can be written inside HTML.
Unlike JavaScript expressions, AngularJS expressions do not support conditionals, loops, or exceptions.
Unlike JavaScript expressions, AngularJS expressions support filters.
