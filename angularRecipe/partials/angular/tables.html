Displaying Data in a Table

The ng-repeat directive is perfect for displaying tables.
Displaying tables with angular is very simple:

{{x.Name}} {{x.Country}}

<div ng-app="myApp" ng-controller="demoCustomerCtrl">
      <table>
           <tr ng-repeat="x in names">
                <td>{ {x.Name} }</td>
                <td>{ {x.Country} }</td>
           </tr>
      </table>
</div>

To make it nice, add some CSS to the page:

<style>
      table, th , td {
           border: 1px solid grey;
           border-collapse: collapse;
           padding: 5px;
      }
      table tr:nth-child(odd) {
           background-color: #f1f1f1;
      }
      table tr:nth-child(even) {
           background-color: #ffffff;
      }
</style>

Display with orderBy Filter

{{ x.Name }} {{ x.Country }}

<div ng-app="myApp" ng-controller="demoCustomerCtrl">
      <table>
           <tr ng-repeat="x in names | orderBy : 'Country'">
                <td>{ {x.Name} }</td>
                <td>{ {x.Country} }</td>
           </tr>
      </table>
</div>

Using $even and $odd

{{ x.Name }} {{ x.Name }} {{ x.Country }} {{ x.Country }}

<div ng-app="myApp" ng-controller="demoCustomerCtrl">
      <table>
           <tr ng-repeat="x in names | orderBy : 'Country'">
                <td ng-if="$odd" style="background-color:#F81553">{ {x.Name} }</td>
                <td ng-if="$even">{ {x.Name} }</td>

                <td ng-if="$odd" style="background-color:#F81553">{ {x.Country} }</td>
                <td ng-if="$even">{ {x.Country} }</td>
           </tr>
      </table>
</div>
